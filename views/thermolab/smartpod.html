인프라 구조도

스마트팟 시스템은 AWS 클라우드 기반으로 구축되어 있습니다.


주요 구성 요소:
• AWS IoT Core: 데이터 로거 디바이스와 안전한 MQTT 통신
• Lambda: 실시간 데이터 처리 및 변환
• EC2 + ELB: Spring Boot 기반 API 서버 (로드 밸런싱)
• RDS (PostgreSQL): 메인 데이터베이스
• S3 + CloudFront: 정적 파일 저장 및 CDN
• Route53 + ACM: 도메인 관리 및 SSL 인증서
• CloudWatch: 시스템 모니터링 및 로깅
• GitHub Actions: CI/CD 파이프라인 (무중단 배포)
