프로젝트 기술서

  • GC CELL API 연동 및 데이터 매핑
  • 실시간 오류 처리 및 재시도 로직 구현

느낀 점

Kotlin의 강력한 비동기 처리 기능을 활용하여 다수의 데이터 소스를 효율적으로 처리할 수 있었습니다. 또한 실시간 데이터 처리에서의 오류 처리와 복구 전략의 중요성을 깊이 이해하게 되었습니다.
