프로젝트 개요

모두가 행사를 편하게 만들고 개최하는 시스템

기술 스택

Java Spring Boot MySQL Thymeleaf JavaScript

기여도

개발팀 구성원 (80%)

프로젝트 정보

서비스 종료

팀 구성

5명 (기획 2명, 개발 2명, 디자인 1명)

프로젝트 이미지

서비스 메인 페이지
행사 생성 및 관리
관리자 대시보드
모바일 반응형 화면

프로젝트 기술서

주요업무

  • 행사 생성 및 관리 플랫폼
  • 참가자 등록 및 관리
  • 결제 시스템 연동
  • 행사 홍보 및 마케팅 도구
  • 실시간 알림 및 소통 기능

상세 업무

  • Spring Boot 기반 RESTful API 서버 개발
  • MySQL 데이터베이스 설계 및 최적화
  • Thymeleaf를 활용한 동적 웹 페이지 개발
  • PG사 결제 모듈 연동 (KG이니시스, 토스페이먼츠)
  • 행사 관리 및 참가자 관리 시스템 구축
  • 실시간 알림 기능 (WebSocket 활용)

성과

  • 2년간 안정적인 서비스 운영
  • 월평균 500여 개의 행사 생성
  • 누적 사용자 10,000명 돌파

느낀 점

팀 프로젝트에서 백엔드 개발을 담당하며 프론트엔드 팀원들과의 협업과 API 설계의 중요성을 배웠습니다. 특히 결제 시스템과 같은 민감한 기능을 다루며 보안과 안정성에 대한 깊은 이해를 얻을 수 있었습니다.
