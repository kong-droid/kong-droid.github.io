프로젝트 개요

다양한 힐링 프로그램을 소개하고, 간편 오프라인 결제로 온라인 강의를 들을 수 있는 시스템

기술 스택

PHP MySQL JavaScript HTML/CSS Payment API

기여도

개인 프로젝트 (100%)

프로젝트 기술서

주요업무

  • 힐링 프로그램 소개 및 카탈로그
  • 온라인 강의 플랫폼
  • 오프라인 결제 시스템 연동
  • 사용자 학습 진도 관리
  • 강사-수강생 소통 기능

상세 업무

  • PHP 기반 웹 애플리케이션 개발
  • MySQL 데이터베이스 설계 및 구현
  • 결제 API 연동 (KCP, 이니시스)
  • 온라인 강의 스트리밍 시스템 구축
  • 사용자 인증 및 권한 관리 시스템
  • 학습 진도율 추적 및 관리 기능

성과

  • 6개월간 성공적인 이벤트 운영
  • 1,000여 명의 참가자 등록
  • 안정적인 결제 시스템 운영 (오류율 0.5% 미만)

느낀 점

PHP와 MySQL을 활용한 웹 개발의 기초를 다지며, 결제 시스템과 같은 중요한 기능을 안정적으로 구현하는 경험을 했습니다. 특히 사용자 경험을 고려한 UI/UX 설계와 데이터베이스 최적화의 중요성을 배웠습니다.
