프로젝트 기간

2019.01 ~ 2021.12

프로젝트 개요

DDP에 위치한 어린이 놀이터로 티켓 예약, 이벤트 안내, 어린이 놀이 범위 리포트를 보여주는 웹 프로젝트입니다.

기술 스택

Java 8 JavaScript(ES5) SQL JSP SpringFramework MyBatis JQuery Linux MariaDB

기여도

개인 기여도 70%

3명 (운영팀 2명, 개발팀 1명)

디키디키 홈페이지 메인
티켓 예약 시스템
이벤트 관리 페이지
놀이 범위 리포트

프로젝트 기술서

주요업무

  • 웹사이트 개발 및 유지보수
  • 티켓 예약 시스템 구현
  • 이벤트 관리 기능 개발
  • 어린이 놀이 범위 리포트 시스템 구축
  • DDP 어린이 놀이터 전용 웹 서비스 제공
  • 실시간 예약 현황 관리 시스템

상세 업무

  • Spring Framework 기반 MVC 패턴 웹 애플리케이션 개발
  • MyBatis를 활용한 데이터베이스 연동 및 쿼리 최적화
  • JSP와 JQuery를 활용한 동적 웹 페이지 구현
  • 실시간 티켓 예약 시스템 및 좌석 관리
  • 어린이 놀이 활동 데이터 수집 및 분석 리포트
  • Linux 서버 환경에서의 배포 및 운영

성과

  • 약 3년간 안정적인 서비스 운영
  • DDP 어린이 놀이터의 디지털 서비스 혁신
  • 티켓 예약부터 놀이 관리까지 원스톱 서비스 구현
  • 월평균 1,000건 이상의 티켓 예약 처리

느낀 점

첫 번째 상용 웹 서비스 개발 경험으로 Spring Framework의 기본기를 탄탄히 다질 수 있었습니다. 실제 사용자들이 이용하는 서비스를 개발하며 사용성과 안정성의 중요성, 그리고 지속적인 유지보수의 필요성을 깊이 이해하게 되었습니다.
